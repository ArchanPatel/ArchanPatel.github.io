# Expo Portfolio App (scaffold)

This is a minimal Expo + React Native (TypeScript) scaffold generated for you.
It uses headless component architecture (hooks for logic, UI components for presentation).

Files of interest:
- src/components/headless/useProjects.ts
- src/components/headless/useResumeHighlights.ts
- src/components/ui/*
- src/screens/*

To run:
1. Install Expo CLI / ensure you have `expo` and required SDKs.
2. Run `npm install` (or `yarn`) then `npx expo start`.
